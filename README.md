# Mental Health Monitoring System

## 📌 Project Description
This project analyzes social media text to detect mental health conditions using Machine Learning.

## 🛠️ Technologies Used
- Python
- Machine Learning
- Flask (if used)
- HTML/CSS
- SQLite

## 📂 Project Structure
- app.py → Main application
- model.py → ML model training
- dataset.csv → Dataset
- templates/ → Frontend UI
- users.db → Database

## ▶️ How to Run
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python app.py

## 📊 Features
- User input text analysis
- Mental health prediction
- Simple UI
